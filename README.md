# Customer Order Analysis
 
## Project Overview 
This project analyzes customer order data to understand purchasing behavior, 
identify top performing product categories, evaluate customer segments, 
and measure the effectiveness of acquisition channels.
 
## Business Objective 
The objective of this project is to identify customer purchasing trends, 
determine the most popular product categories, discover cities with the highest order
volume, analyze customer segmentation, and evaluate the most effective customer 
acquisition channels using Microsoft Excel.
 
## Dataset 
The dataset contains customer order records such as order date, region, customer segment,
acquisition channel, product category, item purchased, order value, shipping fee, and delivery status.
 
## Tools Used 
- Microsoft Excel 
- Excel formulas 
- PivotTable or summary analysis
- Power Query 
- Excel dashboard 
- GitHub documentation
 
## Analysis Process 
1. Reviewed the raw customer order dataset.
2. Cleaned duplicate, missing, and inconsistent data.
3. Created calculated fields and summary tables.
4. Built PivotTables and visual dashboards.
5. Analyzed customer purchasing patterns and business performance.
6. Documented findings and business recommendations.

## Key Insights 
- Furniture is the product category with the highest number of customer orders, 
indicating strong customer demand and making it the primary revenue driver.
- Bandung and Medan recorded the highest number of customer orders, suggesting 
these cities represent the strongest markets and should be prioritized for business 
expansion and marketing activities.
- The Corporate customer segment generated the highest order volume, showing that 
business customers contribute significantly to overall sales performance.
- The Event acquisition channel brought in the largest number of customers, 
demonstrating that event-based marketing campaigns are highly effective in attracting and converting new customers.
